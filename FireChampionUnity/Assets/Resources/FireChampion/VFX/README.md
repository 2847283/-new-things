# Fire Champion VFX Assets

This folder contains project-owned transparent PNG effects used by `FireChampionVfxAssets` and `FireChampionVfxRenderer`.

Runtime paths:

- `Resources.Load<Texture2D>("FireChampion/VFX/vfx_hit_sweet")`
- `Resources.Load<Texture2D>("FireChampion/VFX/vfx_hit_solid")`
- `Resources.Load<Texture2D>("FireChampion/VFX/vfx_smash_trail")`
- `Resources.Load<Texture2D>("FireChampion/VFX/vfx_dash_burst")`
- `Resources.Load<Texture2D>("FireChampion/VFX/vfx_trick_spin")`
- `Resources.Load<Texture2D>("FireChampion/VFX/vfx_score_burst")`

These files are transparent 256x256 PNGs generated as original project art on 2026-06-27. They do not change gameplay rules, hit boxes, timing, scoring, AI, or character stats.
