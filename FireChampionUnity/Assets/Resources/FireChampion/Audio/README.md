# Fire Champion Audio Assets

This folder contains project-owned short WAV cues used by `FireChampionAudioAssets`.

Runtime paths:

- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_ultimate")`
- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_skill")`
- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_serve")`
- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_hit_normal")`
- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_hit_smash")`
- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_score_left")`
- `Resources.Load<AudioClip>("FireChampion/Audio/sfx_score_right")`

These files are original generated WAV cues created for this project on 2026-06-27. They are feedback-only assets and do not change gameplay rules, timing, scoring, AI, collision, stats, or input handling. If a clip is missing, the game falls back to the existing synthesized tone path.
