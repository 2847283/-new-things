# Fire Champion UI Resources

These PNG assets are original procedural drawings generated for this project on 2026-06-18.

They are not downloaded from the web and do not use third-party art.

The project-level source and future asset policy is recorded in `Assets/Resources/FireChampion/ASSET_SOURCES.md`.

## Files

- `logo_fire_champion.png`: menu logo mark.
- `role_core.png`: CORE role icon.
- `role_dash.png`: DASH role icon.
- `role_heavy.png`: HEAVY role icon.
- `role_trick.png`: TRICK role icon.
- `court_dojo.png`: dojo court badge.
- `court_rooftop.png`: rooftop court badge.
- `court_future.png`: future court badge.

## Import Use

Runtime path prefix:

`Resources.Load<Texture2D>("FireChampion/UI/<file_without_extension>")`

These assets are cosmetic and UI-only. They must not alter character strength or gameplay stats.
